# FlyRank Capstone

This repository is the starting point for the FlyRank capstone project and serves as the workspace for setup, planning, and iteration.

## Stack
- Node.js (LTS)
- Git
- VS Code
- Claude Code

## Getting started
1. Install Node.js, Git, and VS Code.
2. Clone this repository to your local machine.
3. Open the project in VS Code and review the project guidance in CLAUDE.md.
4. Start building your capstone work from this baseline.

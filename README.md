# FlyRank Capstone

Welcome to the FlyRank capstone repository. This project serves as the starting point for building, documenting, and iterating on your capstone work in a clean and organized workspace.

## Overview
This repository is designed to help you:
- set up your development environment quickly,
- organize project files and documentation,
- follow a consistent workflow for commits and collaboration.

## Tech Stack
- Node.js (LTS)
- Git
- VS Code
- Claude Code

## Getting Started
1. Install Node.js, Git, and VS Code.
2. Clone this repository to your local machine.
3. Open the project in VS Code.
4. Review the guidance in CLAUDE.md for project conventions.
5. Start building your capstone work from this baseline.

## Project Structure
- README.md: project overview and setup guidance
- CLAUDE.md: repository conventions and workflow notes
- LICENSE: open-source license for the project
- docs/: project planning and supporting notes

## Goals
The main goal of this repository is to provide a solid foundation for the capstone project while keeping the setup simple and organized.

## Workflow
- Create or update files as needed for your project work.
- Keep documentation clear and up to date.
- Use concise commit messages that describe the change.
- Push updates regularly to keep the repository current.
